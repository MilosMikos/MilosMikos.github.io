The kit aims for openneness and efficiency in it's software choices. Though some choices had to be made. The kit was mainly done using photoshop. Most translations were made on photopea which has the best .psd parser of all softwares and I wish it was more open. Illustrations were made using inkscape, and a bit of krita. The code here was made by hand, with knowledges about html, css and programming, but also with some AI. Take it like you want, but remember the goal is the result for code. I used chatGPT which often told me dumb things, and Claude who wanted to make a GIGA corporate website. And helped fight agains't my mental meltdown by somehow fixing my spaghetti. I should mention also tf2classfied and open fortress teams especially the webdevs, because I yanked ideas and stuff you might recognise.
